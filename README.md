# yim-ui

## Installation

For node.js, you can use this command to install:

    npm install yim-ui

## Usage

你可以引入整个 YimUi:

```JavaScript
import YimUi from 'yim-ui'
import 'yim-ui/dist/style.css'
Vue.use(YimUi)
```

你也可以按需引入部分组件:(然后进行全局或者局部注册)

```JavaScript
import { YmButton } from 'yim-ui';
```
