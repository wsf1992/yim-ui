# yim-ui

## Installation

For node.js, you can use this command to install:

    npm install yim-ui

## Usage

You could use like this:

```JavaScript
import YimUi from 'yim-ui'
Vue.use(YimUi)
```

If you use node.js, you should require the module first:

```JavaScript
import queue from 'js-queue';
```
